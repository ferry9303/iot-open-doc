# IoT 概念参考

## 枚举数据 ENUM

### Device.feature

| 设备特性值          | 说明               |
|------------------- | -------------------|
| SHOWER             | 淋浴设备             |

### Device.status

| 设备状态值           | 说明               |
|------------------- | -------------------|
| AVAILABLE          | 可使用的           |
| USING              | 正在使用中          |
| ABNORMAL           | 异常               |
| LOCKED             | 锁定               |
| OCCUPIED           | 占用               |

### Occupation.status

| 占用状态值           | 说明               |
|------------------- | -------------------|
| OCCUPIED           | 已占用              |
| OCCUPY_FAILURE     | 占用失败             |
| RELEASED           | 已释放              |
| RELEASE_FAILURE    | 释放失败             |
